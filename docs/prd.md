# 需求文档

## 1. 应用概述

**应用名称**：趣种地

**应用描述**：趣种地是海南地区极具影响力的亲子自然教育与研学实践平台微信小程序，专注为3–14岁青少年及家庭提供高品质、沉浸式、多元化的户外成长体验。依托海南得天独厚的自然生态与热带农业资源，打造集自然科普、农耕实践、文化传承、素质拓展、亲子互动于一体的综合性营地教育体系。

## 2. 用户与使用场景

**目标用户**：3–14岁青少年家庭、对亲子自然教育与研学实践感兴趣的大众家庭

**核心使用场景**：
- 微信一键登录或手机号/邮箱登录
- 了解趣种地平台介绍与营地项目
- 浏览自然课堂与亲子运动两大分类项目
- 查看自然课堂详情（可种植产品、地块类型、合作模式）
- 查看亲子运动详情（传统文化运动项目）
- 在线预约自然课堂或亲子运动项目，使用优惠券和积分抵扣
- 查看我的订单及订单状态，取消未开始的预约，申请退款
- 自然课堂作物成熟后选择邮寄或现场采摘，邮寄需支付费用
- 查看种植进度跟踪与实时监控画面
- 查看物流状态
- 完成体验后对项目打分和评价，上传图片
- 查看个人中心信息、我的评价、积分余额
- 接收微信订阅消息提醒
- 查看营地地图位置并导航
- 提交意见建议
- 查看联系方式与营业信息
- 生成分享海报并保存到相册
- 商家登录后台管理订单、更新种植进度、确认预约、填写物流单号

### 3.1 登录/注册页面

**页面说明**：用户登录或注册入口

**功能说明**：
- 提供微信一键登录按钮，静默登录获取openid

- 登录成功后跳转至首页Tab

## 4. 业务规则与逻辑

### 4.1 用户注册与登录规则

- 支持微信一键登录（静默登录获取openid）
- 支持手机号登录（输入手机号和验证码）
- 支持邮箱登录（输入邮箱和密码）
- 首次登录用户自动完成注册，创建用户账号
- 用户信息存储至Supabase数据库，包括：用户ID、openid、手机号、邮箱、昵称、头像、注册时间、积分余额

### 4.2 数据存储规则

- 意见建议表单提交后，数据保存至Supabase数据库
- 存储字段包括：姓名、电话、意见/建议内容、提交时间
- 预约订单提交后，数据保存至Supabase数据库
- 自然课堂订单存储字段包括：订单编号、用户ID、项目类型、具体项目、地块类型、预约日期、参与人数、联系人姓名、联系人电话、使用的优惠券、使用的积分、订单金额、提交时间、种植进度、收获状态、支付状态、退款状态、物流单号、物流状态
- 亲子运动订单存储字段包括：订单编号、用户ID、项目类型、具体项目、预约日期、参与人数、联系人姓名、联系人电话、使用的优惠券、使用的积分、订单金额、提交时间、预约状态、体验状态
- 评价数据存储字段包括：评价ID、订单编号、用户ID、项目类型、具体项目、评分、评价内容、评价图片、提交时间
- 优惠券数据存储字段包括：优惠券ID、优惠券名称、优惠金额、使用条件、有效期、发放数量、已使用数量
- 用户优惠券关联表存储字段包括：用户ID、优惠券ID、领取时间、使用状态、使用时间
- 积分记录存储字段包括：用户ID、积分变动类型（获得/使用）、积分数量、关联订单编号、变动时间
- 分享海报图片存储至Supabase Storage，存储字段包括：海报ID、用户ID、项目类型、订单编号（如适用）、图片URL、生成时间

### 4.3 视觉风格规则

- 整体采用田园自然风格，绿色主题
- 项目Tab、自然课堂详情页、亲子运动详情页、预约页面、订单页面使用田园自然风格UI
- 首页保持宇宙星空主题（深色背景+星星闪烁动画+彩色浮动光点）
- 内容卡片使用白色半透明样式
- 可种植产品列表中，辣椒和土豆前展示合适的MDI图标，确保排版整齐

### 4.4 自然课堂合作模式

- 与本地农庄农户合作
- 平台抽取10%佣金
- 成熟后用户可选择是否邮寄并付款

### 4.5 订单状态流转规则

**自然课堂订单状态流转**：
- 种植进度：待种植 → 已播种 → 生长中 → 已成熟
- 收获状态：待收获 → 待邮寄/现场采摘 → 已邮寄（仅邮寄）
- 支付状态：未支付 → 已支付（仅邮寄）
- 退款状态：无 → 退款中 → 已退款（仅已支付邮寄费订单）
- 订单可取消条件：种植进度为「待种植」
- 取消后订单状态更新为「已取消」，默认不在订单列表展示，需切换至「已取消」Tab查看

**亲子运动订单状态流转**：
- 预约状态：待确认 → 已确认 → 已完成/已取消
- 体验状态：未体验 → 已体验
- 订单可取消条件：预约状态为「待确认」
- 取消后预约状态更新为「已取消」，默认不在订单列表展示，需切换至「已取消」Tab查看

### 4.6 邮寄与现场采摘规则

- 自然课堂作物成熟后，用户可选择邮寄或现场采摘
- 选择邮寄需支付邮寄费用（含邮费和包装费），通过微信支付完成付款
- 选择现场采摘无需额外费用，收获状态直接更新为「现场采摘」
- 商家填写物流单号后，用户可查看物流状态

### 4.7 评价规则

- 用户完成体验后（订单状态为「已完成」且体验状态为「已体验」）可对项目进行评价
- 评价包含评分（1-5星）、评价内容、评价图片（最多9张）
- 提交评价后，用户获得积分奖励（每次评价获得10积分）
- 评价展示在项目详情页，按时间倒序排列

### 4.8 消息通知规则

- 使用微信订阅消息功能
- 通知场景包括：
  - 预约到期前1天提醒用户
  - 自然课堂作物成熟提醒用户选择邮寄或现场采摘
  - 商家确认预约后通知用户
  - 物流单号填写后通知用户

### 4.9 优惠券规则

- 商家可在后台发放优惠券，设置优惠券名称、优惠金额、使用条件、有效期、发放数量
- 用户在预约页面可选择可用优惠券，每个订单仅可使用一张优惠券
- 优惠券使用后状态更新为「已使用」
- 优惠券过期后不可使用

### 4.10 积分规则

- 用户完成订单后获得积分（每完成一个订单获得订单金额的1%积分）
- 用户提交评价后获得积分（每次评价获得10积分）
- 用户在预约页面可选择使用积分抵扣订单金额（100积分=1元）
- 积分使用后从用户积分余额中扣除

### 4.11 商家后台管理规则

- 商家使用账号密码登录后台
- 商家可查看所有用户订单，筛选订单状态
- 商家可更新自然课堂订单的种植进度
- 商家可确认亲子运动订单的预约状态
- 商家可填写已邮寄订单的物流单号
- 商家可上传项目展示图片

### 4.12 分享功能规则

- 所有页面均支持分享功能
- 分享内容包括页面标题、描述、缩略图

### 4.13 分享海报生成规则

- 用户在项目详情页或订单详情页点击「生成分享海报」按钮
- 前端调用后端Edge Function，传递项目信息或订单信息
- 后端Edge Function使用Canvas/HTML生成PNG格式海报图片
- 海报包含：项目名称、营地logo/名称、小程序码（占位）、用户预约信息或项目亮点介绍
- 生成的海报图片存储至Supabase Storage
- 后端返回海报图片URL至前端
- 前端展示海报图片，用户可点击「保存到相册」按钮将海报保存至相册

### 4.14 订单列表筛选规则

- 订单Tab提供筛选Tab：全部/待确认/进行中/已完成/已取消
- 默认展示「全部」Tab，不包含「已取消」状态订单
- 点击「已取消」Tab，展示历史取消订单列表
- 其他筛选Tab根据订单状态展示对应订单

## 6. 验收标准

1. 打开趣种地小程序，进入登录/注册页面，点击微信一键登录完成登录
2. 进入首页Tab，查看宇宙星空背景效果与平台介绍文字
3. 点击营地项目预览卡片入口，跳转至项目Tab
4. 在项目Tab查看自然课堂与亲子运动两大分类卡片及平均评分
5. 点击自然课堂卡片，进入自然课堂详情页，查看项目介绍、可种植产品（确认辣椒和土豆前展示合适图标）、地块类型、用户评价列表
6. 点击「生成分享海报」按钮，查看生成的海报图片（包含项目名称、营地logo/名称、小程序码、项目亮点介绍），点击「保存到相册」按钮将海报保存至相册
7. 点击「立即预约」按钮，在预约页面填写具体项目、地块类型、预约日期、参与人数、联系人姓名、联系人电话，选择可用优惠券和使用积分抵扣，查看订单金额明细，点击提交预约
8. 切换至订单Tab，确认默认展示「全部」Tab且不包含「已取消」订单，查看刚提交的自然课堂订单，确认订单信息与种植进度
9. 点击订单卡片进入订单详情页，查看种植进度跟踪时间线和实时监控画面
10. 点击「生成分享海报」按钮，查看生成的海报图片（包含项目名称、营地logo/名称、小程序码、用户预约信息），点击「保存到相册」按钮将海报保存至相册
11. 当种植进度为「已成熟」时，点击「选择邮寄」按钮，在邮寄支付页面查看邮寄费用明细，点击微信支付按钮完成付款
12. 返回订单详情页，确认收获状态更新为「待邮寄」，支付状态更新为「已支付」，查看物流单号和物流状态
13. 点击「取消订单」按钮（种植进度为「待种植」时），确认订单状态更新为「已取消」，返回订单Tab，切换至「已取消」Tab查看该订单
14. 返回项目Tab，点击亲子运动卡片，进入亲子运动详情页，查看项目介绍、运动项目列表、用户评价列表
15. 点击「生成分享海报」按钮，查看生成的海报图片，点击「保存到相册」按钮将海报保存至相册
16. 点击「立即预约」按钮，在预约页面填写具体项目、预约日期、参与人数、联系人姓名、联系人电话，选择可用优惠券和使用积分抵扣，查看订单金额明细，点击提交预约
17. 切换至订单Tab，查看刚提交的亲子运动订单，确认订单信息与预约状态
18. 点击订单卡片进入订单详情页，点击「生成分享海报」按钮，查看生成的海报图片，点击「保存到相册」按钮将海报保存至相册
19. 当预约状态为「已完成」且体验状态为「已体验」时，点击「去评价」按钮，在评价页面填写评分、评价内容，上传评价图片，点击提交评价，确认积分余额增加
20. 切换至个人中心Tab，查看用户头像、昵称、手机号、积分余额，点击「我的评价」查看历史评价列表，点击「我的优惠券」查看可用优惠券列表
21. 切换至联系Tab，查看地址、电话、营业时间、开放说明，查看地图展示营地位置，点击「导航」按钮调用微信地图导航功能
22. 商家登录后台管理页面，查看订单管理列表，选择自然课堂订单更新种植进度，选择亲子运动订单确认预约状态，选择已邮寄订单填写物流单号，上传项目展示图片
23. 用户接收微信订阅消息提醒（预约到期前提醒、作物成熟提醒、预约确认通知、物流单号填写通知）
